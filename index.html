Image Searcher API

This api lets you get a list of links to images from imgur.


Usage:

To search for images add /search/{searchterm} to the end of the URL,

If you want to choose the amount of results you want back add ?offset={number}

At the end (max 60), otherwise it defaults to ten

To see what other people searched for go to /latest/ and you get a list of

the ten latest searches.
